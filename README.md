# ggpp
Generate Github Profile Picures
```
Usage: ggpp [OPTIONS]

Options:
  -r, --rainbow            Each square in a new color
  -c, --count <COUNT>      How many profile pictures are to be generated [default: 1]
  -u, --upscale <UPSCALE>  Scale up the image [default: 1]
  -h, --help               Print help
  ```
